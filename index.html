Telco Churn Prediction Dashboard

Tools: Python (Google Colab), Power BI, pandas, scikit-learn, DAX
Dataset: Kaggle - Telco Customer Churn

This project analyzes and predicts customer churn using machine learning. I trained a Random Forest model and visualized results in Power BI.

  • KPIs: Total Customers, Predicted Churn, Churn Rate
  • Segmentation by Contract Service & Contract Type
  • Churn likelihood by customer tenure
  • Forecast: Monthly revenue at risk
  • Churn segmentation: Monthly Charges vs Tenure
  • Customer-level appendix with product outcome

WA Mining Tenement Dashboard

Tools: Power BI, DAX, Visual Design, Data Cleaning
Dataset: WA Government Mining Tenement Dataset

This project explores mining tenement data from Western Australia. I cleaned, transformed, and visualized it using Power BI.

  • Slicers for Tenement Type, Status, Area Held
  • Date parsing and unit conversion
  • Daily and total distribution by tenement type

Web Traffic Analysis Dashboard

Tools: Tableau, Kaggle Sample Data
Dataset: Kaggle - Website Traffic and User Engagement Metrics

This dashboard analyze website user behavior, traffic sources, and conversion performance. This project focuses on identifying high-performing channels and potential areas for optimization.

  • Overall KPIs: Bounce Rate, Conversion Rate, Avg. Session Duration, Total Visits
  • Traffic source breakdown: Organic, Paid, Referral, Social, Direct
  • Visitor segmentation: New vs Returning
  • Focus areas: Improve Paid & Social engagement, increase Referral traffic, encourage retention
